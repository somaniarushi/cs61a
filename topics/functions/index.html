What a Function is Made Of

To start understanding functions, the place to start is to understand all the parts that make up a function! Here's a quick, small function that I'll be referring to throughout this note. 

def f(x):
    print("Hi")
    return x+1

f(3)

A good question to start with –– what's going on here?
